## CRM Update Approval Format

When presenting proposed CRM updates for approval, use this format:

```
Proposed Update — [Account Name]

Field: Close Date
Current: 2026-05-31
Suggested: 2026-06-30
Reason: Procurement phase has not started; current date is unrealistic
        based on stage and recent activity.

[ Approve ]  [ Reject ]  [ Modify ]
```

Present each update individually. Do not batch-apply without per-item approval.
